# Modern Portfolio

A stunning portfolio website with a purple and black color scheme featuring a left sidebar navigation.

## Features

- **Modern Design**: Clean, contemporary design with purple and black color palette
- **Left Sidebar Navigation**: Fixed sidebar with smooth scrolling navigation
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Hover effects, animations, and transitions
- **Portfolio Sections**: Hero, About, Skills, Projects, and Contact sections
- **Contact Form**: Functional contact form with validation
- **Social Links**: Integrated social media links
- **Smooth Animations**: Fade-in effects, parallax scrolling, and typing animations

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through sections using the left sidebar
3. Fill out the contact form to test functionality
4. View the responsive design by resizing your browser

## Customization

### Personal Information
- Update name, title, and contact details in `index.html`
- Replace social media links with your own profiles
- Modify the logo text in the sidebar

### Color Customization
- Update CSS custom properties in `:root` section of `styles.css`
- Adjust gradient colors for different themes

### Content Updates
- Edit project descriptions and technologies
- Add or remove skills in the skills section
- Customize the about section content
